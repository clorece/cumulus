# Cumulus — preset gallery

All 24 wallpaper-paired presets, in cycle order. Each shows the wallpaper + bar; the seven
marked **↓ dashboard** also include a full shot with the dashboard and notification center open.
Wallpapers by [@XilmO@夕末 on pixiv](https://www.pixiv.net/en/users/19389056) — see
[CREDITS](../CREDITS.md).

> `dark` / `light` indicates the scheme mode (light presets flip GTK/Qt/terminals to light too).
